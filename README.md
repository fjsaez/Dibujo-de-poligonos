# Dibujo-de-poligonos

---
## Descripción
Aplicación simple que dibuja un polígono irregular a partir de coordenadas contenidas en un archivo CSV.

## Lenguaje/framework

* Lenguaje: Delphi 10.2 Tokyo, empleando sólo componentes nativos. 
* Framework: VCL.
